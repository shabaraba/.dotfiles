# .dotfiles
dotfiles for me.

# required
if you use WSL2, you should install following:
- genie (for systemd)

# usage
```sh
git clone https://github.com/shabaraba/dotfiles.git
cd dotfiles 
make install
```

# install
Executing [the above command](#usage), you can install as following:
- neovim
- dein
- zplug
